# 🪨📜✂️ Stone Paper Scissor Game

## 🎮 Overview

This is a fun and interactive Stone-Paper-Scissor game built using **HTML, CSS, and JavaScript**. Play against the computer, test your luck, and keep track of your scores!

## ✨ Features

✔️ **Simple & Engaging UI** – Clickable choices with visuals for rock, paper, and scissors.\
✔️ **Randomized Computer Moves** – The computer makes unpredictable choices.\
✔️ **Live Scoreboard** – Tracks both the player’s and computer’s scores.\
✔️ **Instant Feedback** – Displays game results with colored messages.\
✔️ **Reset Option** – Restart the game anytime with a click.

## 📂 Project Files

### 🏗️ `index.html`

📌 Game structure with:

- Title
- Choice buttons (Rock, Paper, Scissors)
- Scoreboard
- Message display & reset button

### 🎨 `style.css`

📌 Styles and layout:

- Clean and modern UI
- Hover effects for interaction
- Responsive design for better accessibility

### 🧠 `script.js`

📌 Game logic and interactions:

- Generates a **random computer choice**
- Compares player and computer choices
- Updates the **scoreboard** dynamically
- Displays game results with **color-coded messages**
- Implements a **reset function**

## 🚀 How to Play

1️⃣ **Open** `index.html` in your browser.\
2️⃣ **Choose** Rock, Paper, or Scissors by clicking on an icon.\
3️⃣ **See Results** – The game announces the winner and updates the score.\
4️⃣ **Reset** the game anytime using the Reset button.
